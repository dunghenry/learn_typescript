# Learn TypeScript

## 1, Install TypeScript Global

```js
npm i -g typescript
```

## 2, Create Project

```js
npm init  -y
```

## 3, Install package dependencies

```js
npm i jest nodemon @types/node @types/jest ts-jest typescript -D
```

## 4, Run watch and compile js

```js
tsc -w index.ts
```

## 5, Create config jest

```js
npx ts-jest config:init
```
